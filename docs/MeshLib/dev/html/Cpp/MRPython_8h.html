For the proper stub file generation, a class must be defined prior to its depending methods. To achieve it, declare the class before the MR_ADD_PYTHON_CUSTOM_DEF block:

  • for simple cases use MR_ADD_PYTHON_CUSTOM_CLASS macro;
  • to customize class declaration or instantiation (custom holder, custom container binding, etc.), use MR_ADD_PYTHON_CUSTOM_CLASS_DECL and MR_ADD_PYTHON_CUSTOM_CLASS_INST_FUNC macros Finally replace the former class declaration within the MR_ADD_PYTHON_CUSTOM_DEF with the MR_PYTHON_CUSTOM_CLASS( class-name ) construction.
